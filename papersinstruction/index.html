Student Research Paper Instructions

BIT Durg - Minor Project Paper Publication Guide

Academic Session: 2025-2026

General Instructions for All Students

1. Your Tasks

  • Abstract (150-250 words)
  • Introduction (1-2 pages)
  • Related Works / Literature Review (2-3 pages with 15-25 references)

2. Professor Will Handle

  • Methodology
  • Results and Discussion
  • Conclusion

3. Overleaf Instructions

  • Create project using IEEE or journal-specific template
  • Add professor as Editor: ankushressci@gmail.com
  • Share the Overleaf link via email/WhatsApp/Discord
  • Use proper LaTeX formatting for citations (BibTeX)

4. Reference Guidelines

  • Use recent papers (2020-2025 preferred)
  • Include at least 15-20 peer-reviewed references
  • Use Google Scholar, IEEE Xplore, Springer, ACM Digital Library
  • Cite properly using BibTeX format
